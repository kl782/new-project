# New Project

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

## Testing
Run tests with: `npm test`

## Project Structure
- `src/`: Source code
- `tests/`: Test files
- `components/`: Reusable React components
- `pages/`: Next.js pages
- `styles/`: CSS styles
- `utils/`: Utility functions
